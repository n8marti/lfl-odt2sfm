# lfl-odt2sfm

- Export text from [LessonsFromLuke](https://scripture-engagement.org/content/lessons-from-luke/) ODT files to [Paratext](https://paratext.org) [SFM](https://ubsicap.github.io/usfm/) files to facilitate translation into other languages.
- Import SFM file text back into properly-formatted ODT files for publication or further manipulation.
